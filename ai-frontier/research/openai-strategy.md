# OpenAI 战略分析 2026

## 关键动作

### 砍掉 Sora (2026年3月)
- Disney 合作进行中突然中止
- Sora 2 最终版本停留在 2025 年 9 月
- **分析**: OpenAI 在做减法。视频模型推理成本极高，商业化远比想象中难。聚焦 ChatGPT + API 收入是理性选择。

### o3 & o4-mini 发布 (2026年4月)
- RL scaling 提升 10x
- o4-mini 更便宜但能力全面超越前代
- **分析**: OpenAI 的 RL 路线在产出成果。o4-mini 的性价比路线说明他们意识到不是所有用户都需要最强模型。

### Codex CLI 开源
- Terminal agent 进入 OpenAI 产品线
- **分析**: 直接对标 [[claude-code]]。开源策略可能吸引社区贡献，但 Claude Code 在 agent 自主性上仍然领先。

### 收购 TBPN
- 持续扩张版图
- 具体影响待观察

## 战略判断

### OpenAI 的困境
1. **营收压力**: 估值 $1000 亿+ 但尚未盈利
2. **产品线过长**: ChatGPT, API, Sora(已砍), DALL-E, Codex, Whisper...
3. **人才流失**: 关键研究人员持续出走（Anthropic, xAI）

### 对 Anthropic 的影响
- OpenAI 砍 Sora = 更聚焦，对 Claude 的竞争压力加大
- Codex CLI 开源 vs Claude Code 付费 = 价格战开始
- 但 Anthropic 在安全和可靠性上有品牌优势

### 你的选择
- 继续 Claude Code 是对的——agent 能力 > 价格
- 关注 Codex CLI 的社区发展，如果生态成熟可以作为备选
- OpenAI 的 API 价格在下降，可以在不需要 agent 的场景用 o4-mini 省钱
